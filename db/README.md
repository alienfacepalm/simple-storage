### Install dbmate
```
brew tap amacneil/dbmate
brew install dbmate
```
On Linux
```
sudo curl -fsSL -o /usr/local/bin/dbmate https://github.com/amacneil/dbmate/releases/download/v1.4.1/dbmate-linux-amd64
sudo chmod +x /usr/local/bin/dbmate
```

### Migration up
```
dbmate up
```
### Migration down
```
dbmate down
```
